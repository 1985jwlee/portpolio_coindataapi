# CoinDataAPI  
실시간 외부 데이터 수집 및 제공을 위한 이벤트 기반 데이터 서버

---

## 1. 프로젝트 개요

CoinDataAPI는 **외부 거래소로부터 실시간 시계열 데이터를 수집·정규화하여 서버 프로세스로 제공하는 데이터 수집 API 서버**이다.  
단순한 API 연동 예제가 아니라, **운영을 전제로 한 실시간 데이터 파이프라인**을 직접 설계·구현한 프로젝트다.

이 프로젝트의 핵심 관심사는 다음과 같다.

- 외부 데이터 소스의 지속적 수집
- 실시간 데이터 처리 및 동시성 제어
- 교체 가능한 데이터 소스 구조
- 서버 프로세스 형태의 안정적인 서비스 제공
- 멀티 플랫폼 배포(Windows / Linux)

---

## 2. 프로젝트 목적

- 외부 거래소 데이터를 **지속적으로 수집**하는 서버 프로세스 구현
- 거래소별 구현을 분리한 **확장 가능한 구조 설계**
- 실시간 시계열 데이터(Kline)를 다루는 처리 흐름 정리
- 라이브러리가 아닌 **운영 가능한 서비스 형태**로 완성

본 프로젝트는 특정 도메인(코인 트레이딩)을 강조하기 위한 것이 아니라,  
**실시간 데이터 수집·처리·제공 시스템을 설계하고 운영할 수 있음을 증명**하는 데 목적이 있다.

---

## 3. 기술 스택

- **C# (.NET Core 기반)**
- 인터페이스 중심 설계
- 이벤트 / 실시간 데이터 처리 구조
- 멀티 플랫폼 빌드 (Windows / Linux)

---

Program

---

## 5. 핵심 구현 내용

### 5.1 실시간 데이터 수집
- 외부 거래소로부터 캔들(Kline) 데이터 수신
- 지속 실행되는 서버 프로세스 형태

### 5.2 인터페이스 기반 확장 구조
- 거래소별 구현을 인터페이스로 분리
- 데이터 소스 교체 및 확장 용이

### 5.3 동시성 및 데이터 안정성
- Rx 기반 동시성 처리 구조 사용
- 실시간 데이터 접근 안정성 확보

### 5.4 운영을 고려한 서비스 형태
- 실행 가능한 바이너리 포함
- 서버 설정 외부화(JSON)

---

## 6. 설계 관점 핵심

- 데이터 수집은 이벤트 흐름으로 처리
- 서비스는 라이브러리가 아닌 프로세스
- 확장은 인터페이스 교체로 해결
- 실시간 시스템에서 동시성 최우선

---

## 7. 빌드 및 실행

빌드 산출물은 Windows / Linux 환경 모두 제공된다.

---

## 8. 한 줄 요약

> CoinDataAPI는 코인 프로젝트가 아니라,  
> **실시간 외부 데이터를 수집·정규화·제공하는 이벤트 기반 데이터 서버**다.
